libgami

A GObject-based library for connections to the Asterisk Manager Interface

General Features

  • written in C using the GLib/ GObject libraries
  • up-to-date and (almost) complete support of the Asterisk Manager API
  • support for both synchronous and asynchronous (non-blocking) operations
  • mapping of action response packages to Glib data types (gboolean, gchar *, GHashTable and GSList)
  • mapping of Asterisk Manager events to the GObject messaging system
  • basic abstraction of different manager API versions
  • support for GObject introspection
  • complete API documentation
  • full support on Linux and Windows

Dependencies

  • GLib >= 2.16
  • GObject >= 2.16
  • GIO >= 2.16

Installing

$ ./configure
$ make
$ make install
